PampamJS

Simple & powerful JavaScript library for building modern web applications.

Instead of learning new paradigm, with PampamJS you can continue using your previous knowledge of HTML, CSS, and JavaScript to create modern web applications.


Getting started Read the documentation



The Web You Know

If you are already know how to code a website, you already know PampamJS. It gives you power to your existing knowledge of HTML, CSS, and JavaScript instead of forcing you to learn new concepts.


Easy Routing

Web application consists of different part of pages (components, if you like). You can create a SPA (single-page application) with advanced routing without additional library. Because it is the core of PampamJS.


Data Binding

The power of modern JavaScript frameworks is the ability to refresh content when changes happen in the real-time. With PampamJS, you can do that too. And it's really fast.


Built for Mobile & Desktop. Built for last.

PampamJS is built with mobile-first approach without compromising more powerful desktop experience. It is performant and supports older Chrome, Firefox, Safari, and Edge versions up to 5 years.
