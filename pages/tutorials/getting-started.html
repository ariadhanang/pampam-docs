Getting Started

Creating a web application using PampamJS is easy, it only takes one minute to setup.

Download pampam.js Download sample project (.zip)


Quick Setup

Let's assume we have our project folder structured like on the image.

  1. Add the downloaded pampam.js (or, use CDN) to index.html script tag. 
    <script type="text/javascript" src="https://pampam.netlify.app/pampam-latest.js"></script>
  2. Add <div> tag with id "root" inside the <body> to mark where our contents will be placed. 
    <div id="root"></div>
  3. Define your app in script.js file. 
    var app = new Pampam("myapp", [
	{ path: "/", templateUrl: "/pages/home.html" },
	{ path: "/about", templateUrl: "/pages/about.html" },
]);

Running on Web Server

Our app requires a web server to run locally. The simplest way is using http-server package from NPM. If you have NodeJS and NPM installed, run this command inside the project folder:

$ npx http-server -p 8888 -c-1 -P http://localhost:8888?

Change 8888 to your desired port (optional).

To run on production server like Apache or Nginx, please read Deployment page.
